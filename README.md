# betterme

Crea el archivo `secrets.properties` en `src/main/resources/secrets` con estas propiedades:

```properties
# JWT
jwt.seed=

# DataSource Configurations
spring.datasource.url=jdbc:mysql://{host}/betterme
spring.datasource.password=
spring.datasource.username=
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
```
